# Feed & Collections Review

## Goal

Expand feed data collection with pagination, then refine freshness, capacity, and personalization strategies before implementation.

## Key Steps

1. Audit current ingestion/storage (`src/hooks/useProductFeedSync.ts`, `supabase/functions/sync-product-feed/index.ts`) and read path (`src/hooks/useFriendFeed.ts`, `src/pages/feeds/FeedPage.tsx`).
2. Design cursor-based pagination flows for Shopify hooks (`useSavedProducts`, `useProductLists`, `useOrders`, `useRecentProducts`, `useFollowedShops`, `useRecommendedProducts`) so sync can pull deeper datasets.
3. Update backend sync handling (including dedupe readability refactor) for larger payloads and note any schema/index updates needed for volume.
4. Recommend freshness, capacity, and personalization enhancements leveraging the richer data set.
